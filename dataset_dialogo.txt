user: Come funziona una funzione ricorsiva?
assistant: Una funzione ricorsiva è una funzione che chiama se stessa. Deve avere un caso base per fermarsi. Per esempio:
def fattoriale(n):
    if n == 0:
        return 1
    return n * fattoriale(n - 1)
Senza il caso base, la ricorsione sarebbe infinita.
